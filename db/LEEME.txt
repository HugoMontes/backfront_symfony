1. Instalar el proyecto con el siguiente comando:
   >php symfony new curso-backfront/backfront_symfony 3.3.2
2. Ejecutar el servidor
   >php bin/console server:run
3. Instalar los siguiente paquetes:
  backfront_symfony>composer require firebase/php-jwt
  >backfront_symfony>composer require knplabs/knp-paginator-bundle
4. Hacer uso de archivos yml para mapear las rutas.
    - Crear la estructura: src/AppBundle/Resources/config/routing
    - Crear backfront_symfony/src/AppBundle/Resources/config/routing/default.yml
    - Crear backfront_symfony/src/AppBundle/Resources/config/routing.yml
    - Editar backfront_symfony/app/config/routing.yml
5. Crear un nuevo bundle para guardar todo lo que tiene que ver con los modelos.
   Es una buena practica para no repetir las entidades en cada bundle.
   Crear el bundle con el siguiente comando:
   backfront_symfony>php bin/console generate:bundle --namespace=BackendBundle --format=yml
   (Presionar Enter para todo)
6. Ingresar a backfront_symfony/app/config/config.yml y eliminar la ruta creada "backend".
7. Crear y configurar las entidades: Mapear las tablas de la BD con el siguiente comando:
   backfront_symfony>php bin/console doctrine:mapping:import BackendBundle yml
8. Cambiar los nombres de las clases de plural a singular.
9. Generar las entidades con el siguiente comando:
   backfront_symfony>php bin/console doctrine:generate:entities BackendBundle
11.Para realizar pruebas ingresar un registro users en la BD con password null
10.Realizar pruebas editar DefaultController
11.Devolver respuestas JSON, Editar DefaultController
